### Instalação: Python

## Faça o download e instale o python no seu computador pelo link:
https://www.python.org/downloads/

## Faça o download e instale o VSCODE ou PCHARM
https://code.visualstudio.com/download
https://www.jetbrains.com/pycharm/download/#section=windows

# Comando para verificar a versão do python, PowerShell do PC:
python --version

# Comando para executar arquivo python, PowerShell do PC:
acesse o diretório do projeto
py .\print.py

# O programa em python é executado linha após linha (sequência).

# VSCODE - acentuação: caso esteja com algum problema, utilize (# -*- coding: utf-8 -*-), no inicio do código.
# VSCODE - a acentuação sairá correta, executando o arquivo python pelo campo TERMINAL.
# VSCODE - no campo SAIDA do TERMINAL, a acentuação, exibirá incorretamente.

# Antes de rodar o projeto, não esqueca de salvar, para que as alterações sejam efetivadas.

# Teclas de atalho: VSCODE
Executa o código - Control + Alt + N 
Salva o projeto - Control + S 

# Comandos terminal: VSCODE
Limpa o conteúdo do terminal - clear ou cls
Cria um ambiente virtual no diretório: python -m venv venv 
Ativa um ambiente virtual no diretório: .\venv\Scripts\Activate.ps1
Executa o app python: py nome_do_arquivo.py
